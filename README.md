# Beach Review App - Oahu, Hawaii

This app allows users to easily view, create, edit, and delete reviews for different beaches in Oahu.

## Install Instructions

execute
 $ bundle install

## Usage

1. Run rake db:migrate
2. Run shotgun
3. Sign Up to start using app.
